# Border
## Shorthand
```css
.border {
    border: solid red 3px;
}
```
ou
```css
.border {
    border-style: dotted;
    border-style: dotted solid;
    border-style: dotted solid double;
    border-style: dotted solid double dashed;
    border-width: 4px 2px 6px 15px;
    border-color: red green blue black;
}
```
